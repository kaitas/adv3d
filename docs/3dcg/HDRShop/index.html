　次世代コンテント研究室では、実空間の照明条件の取得の際に、 Paul Debevec 氏によって開発された HDRShop というソフトウェアを使用しています。


HDRShop の公式ホームページへ行く

そして、この HDRShop のチュートリアル等の日本語訳の作業を、研究開発の一環として行いました。 以下はその成果になります。HDRI作成のご参考になれば幸いです。


　・イントロダクション （Introduction）
　・チュートリアル （Tutorials）
　　　・まずはじめに (Getting Started)
　　　　　　−HDRIのロード、ビュー、操作、保存方法
　　　・カメラカーブ・キャリブレーション (Camera Curve Calibration)
　　　　　　−カメラレスポンス・カーブの理解方法
　　　・HDRI・アセンブリ (HDR Image Assembly)
　　　　　　−標準的なカメラで撮影された露光の異なるイメージの結合による、HDRIの作成方法
　　　・パノラマイメージの変換 (Panoramic Image Transformations)
　　　　　　−様々なレイアウト間でのHDRパノラマの変換方法
　　　・ライト・プローブ・イメージの作成 (Creating a Light Probe Image)
　　　　　　−１つ、または２つの鏡面球からのライト・プローブ・イメージ（HDRパノラマ）の作成方法
　　　・乱反射/鏡面反射 (Diffuse and Specular Convolution)
　　　　　　−乱反射、または鏡面反射カーネルを使ったHDRパノラマの回旋方法
　　　・HDRイメージ編集 (HDR Image Editing)
　　　　　　−HDR Shopを介した、Adobe Photoshopのような既存の画像編集ソフトウェアの用い方
　　　・HDR Shopプラグイン (HDR Shop Plugins)
　　　　　　−HDR Shopにおけるプラグインの使用方法
　　　・HDR Shopプラグインの記述 (Writing an HDR Shop Plugin)
　　　　　　−独自のプラグインの書き方に関して
