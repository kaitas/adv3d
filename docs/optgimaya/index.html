OptGI Render for MAYAについて

　Opt GIRender for Maya は、株式会社オプトグラフによって開発されたラジオシティエンジンを含むオプトグラフラジオシティライブラリを使用して、グローバルイルミネーション（以下、GI）計算をMaya上で実行する、Mayaプラグインパッケージです。

オプトグラフラジオシティライブラリでは、GI計算を照度値（lx）をベースに行うため、照明シミュレーション等をMaya上で行うことが出来ます。

また、オプトグラフラジオシティライブラリは、High Dynamic Range Image（以下、HDRI）によるGI計算もサポートし、パッケージに含まれるHDRIファイルのI/Oプラグインと併用することで、より写実的な表現が可能になります。

更に、オプトグラフラジオシティライブラリのGI計算結果は、レンダリング画像ではなく、シーン中のオブジェクトの照度値を浮動小数点値で持つテクスチャ（以下、イルミナンスマップ）になります。よって、一度計算したイルミナンスマップをカメラパスアニメーションにおいて再利用することで、効率的なアニメーション制作が可能です。
